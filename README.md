## ⚖️ 게임 룰
- 밤 -> 낮 -> 지목 투표 -> 최후의 반론 -> 찬반 투표 ->
    - 밤 : 각 직업 능력 사용 가능 / 마피아만 채팅 가능
    - 낮 : 채팅
    - 지목 투표 : 자기 자신 포함 산 사람만 투표 가능 , 과반수 이상 표 받아야 지목
    - 최후의 반론
    - 찬반 투표 : 찬성/반대 투표 , 과반수 이상 찬성 해야 사형
  
- 직업 능력
  - 마피아 : 밤에 시민 한명 kill 가능
  - 경찰 : 밤에 마피아 여부 조사 가능

- 승리 로직
  1. 마피아가 없으면 시민 승
  2. 시민2=마피아2 - 끝 → 마피아 승
